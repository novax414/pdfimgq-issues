# pdfimgq issue tracker

This repository is used only for bug reports, feature requests, and general feedback related to **pdfimgq**.

## About pdfimgq

`pdfimgq` is a desktop and CLI tool for verifying the technical quality of raster images embedded in PDF documents.

## What this repository is for

Use the **Issues** tab to:

- report bugs
- suggest improvements
- request new features
- provide feedback about usability or documentation

## Important note

This repository is **issues-only** and does not contain the main project source code.

## Before submitting an issue

Please include as much relevant information as possible, for example:

- pdfimgq version
- operating system
- steps to reproduce the problem
- expected behavior
- actual behavior
- screenshots or error messages, if available
